Proyek

Platform JTI-SIMPEN

Mengembangkan aplikasi JTI-SIMPEN full-stack dengan menggunakan framework Laravel dan menggunakan Database MySQL.

Aplikasi JTI-SIMPEN adalah platform manajemen kompensasi berbasis web yang dirancang untuk mempermudah pengelolaan data kompensasi mahasiswa. Fitur utama meliputi:

  • Dashboard interaktif untuk admin dan pengguna.
  • Manajemen data kompensasi dengan CRUD lengkap.
  • Integrasi dengan MySQL untuk penyimpanan data yang aman.
  • Antarmuka responsif menggunakan Laravel dan Tailwind CSS.

Proyek ini selesai dalam 3 bulan dengan fokus pada efisiensi dan keamanan data.

Sistem Survey Kepuasan Pelanggan

Aplikasi berbasis web untuk manajemen survey kepuasan pelanggan dengan dashboard interaktif menggunakan MySQL dan Laravel.

Sistem ini dirancang untuk mengumpulkan dan menganalisis umpan balik pelanggan secara efisien. Fitur utama meliputi:

  • Pembuatan survey dinamis dengan berbagai tipe pertanyaan.
  • Visualisasi data survey melalui grafik interaktif.
  • Integrasi dengan Laravel untuk backend yang kuat.
  • Pengelolaan data menggunakan MySQL dengan performa tinggi.

Proyek ini membantu bisnis meningkatkan kepuasan pelanggan berdasarkan data real-time.

Website Adopsi Kucing

Merancang portofolio responsif dengan desain modern menggunakan HTML, CSS, dan JavaScript.

Website ini bertujuan untuk memfasilitasi adopsi kucing dengan antarmuka yang ramah pengguna. Fitur utama meliputi:

  • Galeri kucing dengan filter berdasarkan usia dan jenis.
  • Formulir adopsi interaktif dengan validasi client-side.
  • Desain responsif menggunakan CSS Grid dan Flexbox.
  • Animasi halus menggunakan JavaScript untuk pengalaman pengguna yang lebih baik.

Proyek ini menunjukkan kemampuan saya dalam desain front-end yang menarik dan fungsional.
